####  ⭐ ***Instructions pour l’exécution :***

1. ***Compilation des fichiers***

```js

javac *.java

```
   
2. ***Démarrage du serveur de traçage***
   
```js

java MessageTrackerServer

```

3. ***Lancement du coordinateur***

```js

java Coordinator

```

4. ***Lancement des processus distribués***  

```js

java Processus1 
java Processus2
java Processus3
java Processus4

```
